## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Update the following files with your information:

- `app/layout.tsx` - Update metadata (title, description)
- `components/Hero.tsx` - Add your name and introduction
- `components/About.tsx` - Add your skills and bio
- `components/Projects.tsx` - Add your projects
- `components/Contact.tsx` - Add your contact links and email
